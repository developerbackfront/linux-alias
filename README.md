# linux-alias

> [!IMPORTANT]
<i>Alias para otimizar trabalhos em comandos Linux</i>

<p>alias p='pwd'</p>
<p>alias ll='ls -alF'</p>
<p>alias la='ls -A'</p>
<p>alias l='ls -CF'</p>
<p>alias copiar='cp'</p>
<p>alias deletar-pasta='rm -i'</p>
<p>alias atualizar-sistemas='sudo apt-get update -y && apt-get upgrade -y'</p>
<p>alias descricao-command='command'</p>
<p>alias cls='clear'</p>

### Git

<p>alias pull='git pull'</p>
<p>alias push='git push'</p>
<p>alias stash='git stash'</p>

### Support

<p>alias chkspeed='wget http://speedtest.tele2.net/1GB.zip -O /dev/null'</p>
<p>alias clearswap='sudo swapoff -a && sudo swapon -a'</p>
<p>alias timestamp='date +%Y-%m-%dT%H-%M-%S'</p>
